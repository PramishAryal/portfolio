Pong

So to learn game development (using actual graphics this time), I stumbled upon HandMade Hero, a live-stream done by Casey Muratori, where he is developing a game from scratch using c++ (well c-style c++). What he uses is just the win32 api and to me that was just mind blowing. No engines, no frameworks, no libraires, just plain old c++ (again c-style c++) and nothing more. This was the best thing I could've ever stumbled upon. So I watched him upto episode 8 or so. And then decided to code a pong game with the knowledge I gained.

This made me learn two new things. One, I loved making games and two I still do not have enough grasp over c++. As I still can't get through episode 15. Things got big brain too fast for me, at least for now. I will return back to handmade yet again when I'm wiser or just when it clicks for me. So till then its time to learn a graphics library like SDL or even OpenGL.
